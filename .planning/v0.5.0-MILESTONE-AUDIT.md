## Architectural Notes

- ALBUM_METADATA genreTags are per-provider (not merged). Only explicit GENRE requests trigger GenreMerger. Intentional by design.
- ARTIST_TIMELINE is the first composite type — engine auto-resolves sub-types and excludes them from caller results.
- ProviderChain.resolveAll() is generic but only GENRE uses it via MERGEABLE_TYPES.
